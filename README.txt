האתר של רוני · ליווי שינה והתפתחות לתינוקות
=================================================

איך פותחים את האתר
------------------
פשוט לחיצה כפולה על הקובץ  index.html  והוא נפתח בדפדפן. זה הכול — אין צורך בהתקנה.

מבנה הקבצים
-----------
  index.html         – עמוד הבית (אודות רוני + תקציר השירותים)
  services.html      – מה אני מציעה (ליווי שינה / התפתחותי / עיסוי תינוקות)
  certificates.html  – הכשרה ותעודות
  contact.html       – יצירת קשר (טופס שפותח וואטסאפ)
  assets/style.css   – העיצוב (צבעים, גופנים, פריסה)
  assets/script.js   – תפריט נייד + הטופס + אנימציות

מה כדאי לעדכן לפני שמראים את האתר ללקוחות (חשוב!)
--------------------------------------------------
1) מספר וואטסאפ:
   בקובץ  assets/script.js  בשורה הראשונה שמתחת לכותרת:
       window.RONI_WHATSAPP = "972500000000";
   החליפו במספר של רוני בפורמט בינלאומי: 972 ואז המספר בלי ה-0 בהתחלה.
   דוגמה: לנייד 050-123-4567  ->  "972501234567"

2) טלפון ואימייל:
   בקובץ  contact.html  חפשו   tel:+972500000000   ו-   roni@example.com
   והחליפו בפרטים האמיתיים (מופיע גם בפוטר של כל העמודים).

3) אינסטגרם:
   חפשו  https://instagram.com/  בכל הקבצים והחליפו בקישור האמיתי.

4) תמונה של רוני:
   שימו קובץ תמונה בשם  roni.jpg  בתוך התיקייה  assets
   ואז ב-index.html חפשו את ההערה "החליפו את ה-div הזה בתמונה אמיתית"
   והחליפו את ה-<div class="placeholder">...</div> בשורה:
       <img src="assets/roni.jpg" alt="רוני, יועצת שינה לתינוקות">

5) תעודות:
   בקובץ  certificates.html  עדכנו את שמות התעודות, "שם המוסד המכשיר · שנה"
   ואת הטקסטים — לפי ההכשרות האמיתיות של רוני.

6) אזור פעילות:
   בקובץ  contact.html  חפשו "כתבו כאן את האזור" והחליפו (למשל: "מרכז הארץ והסביבה").

איך מעלים את האתר לאינטרנט (כשרוצים)
------------------------------------
האתר סטטי לחלוטין, אז אפשר להעלות אותו בחינם לשירותים כמו:
  • Netlify  (פשוט גוררים את התיקייה לאתר netlify.com)
  • GitHub Pages
  • Cloudflare Pages
ואפשר לחבר דומיין אישי (למשל roni-sleep.co.il).

הצבעים והגופנים
---------------
כל הצבעים מוגדרים במקום אחד: בתחילת הקובץ  assets/style.css  בבלוק שנקרא  :root
אפשר לשנות שם צבע אחד וכל האתר מתעדכן.
