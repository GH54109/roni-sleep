מבנה האתר
---------
האתר בנוי כעמוד אחד ארוך עם גלילה (single page), לפי מסמך המותג:
  Hero → נעים להכיר → למה Baby Vibes → מה אני מציעה →
  סדנאות (0–3 / 3–6) → המלצות → שאלות נפוצות → יצירת קשר.

קבצים:
  index.html         – כל האתר (עמוד אחד)
  assets/style.css   – העיצוב (צבעים, גופנים, פריסה) — הכול בבלוק :root למעלה
  assets/script.js   – תפריט נייד, אקורדיון שאלות, טופס-וואטסאפ, אנימציות
  assets/logo.jpg    – לוגו (אייקון שיתוף / apple-touch)

השפה: שקט, אוויר, חום. שמנת/לבן/בז'/חום רך/מרווה. פחות הוא יותר.
המשפט המוביל: "כשההורה מרגיש בטוח יותר – גם התינוק מרגיש בטוח יותר."
תפקיד ראשי: ליווי התפתחותי + עיסוי תינוקות. ייעוץ שינה = ידע משלים בלבד.
